# 电商客服工作台 UI 改版设计

## 目标

把当前偏普通聊天页的前端，改成更像电商大厂内部客服工作台的界面。风格参考亚马逊运营后台：深色顶部导航、黄色强调色、清晰的数据面板、紧凑的表格和稳定的操作区。

## 设计方向

- 页面整体从“聊天页面”升级为“客服工作台”。
- 顶部使用深蓝/石墨色导航，营造后台系统感。
- 主要操作按钮使用暖黄色强调，提示用户这是关键操作。
- 左侧保留会话设置、快捷话术和数据查询入口。
- 中间保留聊天主流程，用于练习 `/chat` 接口。
- 列表结果仍然在聊天区输出，避免改动过多后端逻辑。

## 保留的功能

- `POST /chat`：发送客服对话。
- `GET /complaints`：查看投诉记录。
- `GET /orders`：查看订单列表。
- `GET /logistics`：查看物流列表。

## 文件分工

- `web/index.html`：负责页面结构，比如顶部栏、侧边栏、聊天区、按钮。
- `web/styles.css`：负责页面视觉，比如颜色、间距、卡片、表格、响应式布局。
- `web/app.js`：负责交互逻辑，比如点击按钮、请求接口、渲染聊天气泡和表格。

## 初学者理解重点

HTML 像房子的骨架，决定页面有什么区域。

CSS 像装修，决定这些区域长什么样。

JavaScript 像工作人员，负责点击按钮后去请求后端，再把结果显示到页面上。
